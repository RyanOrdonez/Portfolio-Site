# Applied Data Science Capstone

Coursework from the IBM Applied Data Science Professional Certificate. This repository contains the modular capstone project focused on SpaceX launch data collection, wrangling, and exploratory analysis.

> **Note:** A more complete, polished version of this project -- including predictive modeling, a Plotly Dash dashboard, and Folium visualizations -- is available in the [spacey-capstone](https://github.com/your-username/spacey-capstone) repository.

## Modules

### Module 1: Data Collection
- **API Data Collection** -- Gathering SpaceX launch data via the SpaceX REST API
- **Web Scraping** -- Extracting Falcon 9 launch records from Wikipedia using BeautifulSoup
- **Data Wrangling** -- Cleaning, transforming, and preparing the dataset for analysis

### Module 2: Exploratory Data Analysis
- **Data Visualization** -- Visual exploration of launch outcomes, payload relationships, and trends using Matplotlib and Seaborn
- **SQL Analysis** -- Querying the dataset to answer targeted questions about launch history and success factors

### Module 3: Interactive Visual Analytics and Predictive Modeling
- *Not yet completed in this repository*

## Technologies Used

- Python, SQL
- Pandas, NumPy, Matplotlib, Seaborn
- BeautifulSoup, Requests
- Jupyter Notebooks
