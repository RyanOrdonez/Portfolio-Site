# Algorithms and Data Structures

Coursework from the Computer Science Fundamentals track of the MS in Data Science at University of Colorado Boulder.

## Courses

| Course | Description |
|--------|-------------|
| Searching, Sorting, and Indexing | Fundamental algorithms, complexity analysis, hash tables, and indexing |
| Trees and Graphs | Binary trees, heaps, graph traversal, shortest paths, and spanning trees |
| Dynamic Programming and Greedy Algorithms | Optimization techniques, memoization, and greedy strategies |

## Technologies Used

Python, Jupyter Notebooks
