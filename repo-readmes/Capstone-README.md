# Data Science Capstone -- Course Notes

**DTSA 5510 / 5841: Unsupervised Algorithms and IBM Capstone Project**

## Overview

This repository contains course overview notebooks and reference materials from the capstone track of the MS in Data Science program at the University of Colorado Boulder. It serves as a personal reference for unsupervised learning algorithms including K-Means, Hierarchical Clustering, DBSCAN, PCA, t-SNE, and NMF.

> **Note:** The completed capstone project is available in the [spacey-capstone](https://github.com/RyanOrdonez/spacey-capstone) repository.

## Contents

- `DTSA_5510_Course_Overview.ipynb` -- Reference notebook covering unsupervised learning algorithms across 8 weeks of coursework

## Topics Covered

- K-Means Clustering
- Hierarchical Clustering and Dendrograms
- DBSCAN
- Principal Component Analysis (PCA)
- t-SNE
- Non-Negative Matrix Factorization (NMF)

## Technologies

Python, Scikit-learn, NumPy, Pandas, Matplotlib, Seaborn, SciPy
