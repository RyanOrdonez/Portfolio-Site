# Databases and SQL

Coursework from the Data Engineering track of the MS in Data Science at University of Colorado Boulder.

## Courses

| Course | Description |
|--------|-------------|
| Relational Database Design | ER modeling, normalization, schema design, and relational algebra |
| SQL Core | Queries, joins, subqueries, aggregation, and data manipulation |
| Advanced Database Topics | Indexing, transactions, stored procedures, and query optimization |
