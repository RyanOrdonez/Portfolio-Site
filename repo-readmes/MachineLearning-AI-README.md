# Machine Learning and Artificial Intelligence

Coursework from the Machine Learning and Artificial Intelligence track of the MS in Data Science at University of Colorado Boulder.

## Courses

| Course | Description |
|--------|-------------|
| Data Mining Pipeline | Data preprocessing, transformation, and evaluation frameworks |
| Data Mining Methods | Classification, clustering, and association rule techniques |
| Data Mining Project | Applied data mining on a real-world dataset |
| Supervised Learning | Regression, classification, model evaluation and selection |
| Unsupervised Algorithms | Clustering, dimensionality reduction, anomaly detection |
| Deep Learning | Neural networks, CNNs, RNNs, and training strategies |
| Generative AI | Generative models, transformers, and prompt engineering |

**Notebooks:** 14 total across all courses.

> **Note:** Standalone project repositories exist for key projects completed during this track.

## Technologies Used

Python, Scikit-learn, TensorFlow/Keras, Pandas, NumPy, Matplotlib, Jupyter Notebooks
