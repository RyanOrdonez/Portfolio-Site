# Professional Skills for Data Science

Coursework from the Professional Skills track of the MS in Data Science at University of Colorado Boulder.

## Courses

| Course | Description |
|--------|-------------|
| Data Science as a Field | Overview of the data science landscape, roles, and workflows |
| Cybersecurity for Data Science | Security principles, threat modeling, and data protection practices |
| Ethics in Data Science | Bias, fairness, privacy, and responsible AI considerations |
| Data Visualization | Design principles, exploratory and explanatory visualization techniques |

Includes a **Data Visualization final project notebook**.

## Technologies Used

Python, Matplotlib, Seaborn, Plotly, Jupyter Notebooks
