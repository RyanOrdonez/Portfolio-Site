# Software Architecture and Statistical Modeling

Coursework from the Software Architecture and Statistical Modeling tracks of the MS in Data Science at University of Colorado Boulder.

## Courses

| Course | Description |
|--------|-------------|
| Regression and Classification | Linear models, logistic regression, and discriminant analysis |
| Resampling, Selection, and Splines | Cross-validation, bootstrap, subset selection, and spline methods |
| Trees, SVM, and Unsupervised Learning | Decision trees, random forests, SVMs, PCA, and clustering |
| Software Architecture and Design | Design principles, modularity, and architectural decision-making |
| Architecture Patterns for Big Data | Distributed systems, batch and stream processing architectures |

## Technologies Used

Python, R, Scikit-learn, Jupyter Notebooks
