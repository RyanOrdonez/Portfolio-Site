# Portfolio Site

My personal portfolio website showcasing my data science projects, skills, and experience.

**Live site:** [ryanordonez.github.io/Portfolio-Site](https://ryanordonez.github.io/Portfolio-Site/)

## Features

- **Sidebar navigation** — Home, About, Projects, Skills, Contact, Blog
- **Project showcase** — Highlighted data science and engineering projects
- **Blog** — Posts on AI topics, industry analysis, and technical deep dives
- **Responsive design** — Dark theme, clean layout

## Tech Stack

- HTML, CSS, JavaScript (vanilla — no frameworks)
- GitHub Pages

## Structure

```
docs/
├── index.html          # Main portfolio page
├── blog.html           # Blog listing page
├── blog/               # Blog post pages
├── posts/              # Additional blog posts
└── assets/
    ├── css/            # Stylesheets
    └── js/             # Scripts
```
